Simple Shooting

Unity C# Combat

Fire projectiles from a spawn point

Attach this script to a weapon or player and assign a bullet prefab to fire projectiles in the forward direction with a simple fire rate.

Base snippet


// Simple projectile shooting script.

using UnityEngine;

public class SimpleShooter : MonoBehaviour
{
    public GameObject bulletPrefab;
    public Transform firePoint;
    public float bulletSpeed = 10f;
    public float fireRate = 0.25f; // seconds between shots

    private float nextFireTime = 0f;

    void Update()
    {
        if (Input.GetButton("Fire1") && Time.time >= nextFireTime)
        {
            Shoot();
            nextFireTime = Time.time + fireRate;
        }
    }

    void Shoot()
    {
        if (bulletPrefab == null || firePoint == null) return;

        GameObject bullet = Instantiate(bulletPrefab, firePoint.position, firePoint.rotation);
        Rigidbody2D rb = bullet.GetComponent();

        if (rb != null)
        {
            rb.velocity = firePoint.right * bulletSpeed;
        }
    }
}

Integration Recipe

  1. Create a bullet prefab with a Rigidbody2D and collider (set to trigger or not based on your design).
  2. Attach SimpleShooter to your player or weapon GameObject.
  3. Set Bullet Prefab to your bullet, and Fire Point to a child Transform that indicates the muzzle position.
  4. Press Play and hold Fire1 (default: left mouse or Ctrl) to fire bullets in the firePoint.right direction.

Common Errors

  • Nothing spawns – Check that bulletPrefab and firePoint are assigned in the Inspector.
  • Bullets don't move – Ensure bullet prefab has Rigidbody2D and that gravity scale is set correctly if you want straight shots.
  • Too many bullets – Increase Fire Rate or implement bullet pooling to avoid excessive instantiation.
