Basic Movement Script

Unity C# Movement

Move a GameObject every frame

This snippet shows the simplest way to move a GameObject in Unity. Below you will find an integration recipe, common errors, a code configurator, and an AI helper to adapt it to your own project.

Base snippet


// Example Unity C# script: Move a GameObject every frame.

using UnityEngine;

public class MyFirstScript : MonoBehaviour
{
    public float speed = 5f;

    // Called once per frame
    void Update()
    {
        // Move the GameObject right every frame
        transform.Translate(Vector3.right * speed * Time.deltaTime);
    }
}

Integration Recipe

  1. Create a new C# script in Unity and name it MyFirstScript (or use the configurator below to change the class name).
  2. Copy the code above into the script file and save it.
  3. In the Hierarchy, create a new GameObject or select an existing one that you want to move.
  4. In the Inspector, click Add Component and attach MyFirstScript.
  5. Press Play: the GameObject should move to the right every frame at the speed you set.

Common Errors

  • Script does not appear in “Add Component” list – Make sure the file name and the class name match exactly (e.g., MyFirstScript.cs and public class MyFirstScript).
  • Nothing moves – Check that the script is attached to the correct GameObject, and that speed is greater than 0.
  • Compilation errors – Verify that using UnityEngine; is at the top and there are no missing braces {} or semicolons ;.
